# Gym Website & Fitness Toolkit

This is a comprehensive front-end website project designed for local gyms to establish an online presence. It is built entirely with **HTML, CSS, and vanilla JavaScript**.

The site not only provides essential information about the gym but also includes a suite of **free, publicly accessible fitness tools** for any user, including a Calorie Calculator, BMI Calculator, and various diet and workout plans.

## ‚ú® Key Features

This project is divided into two main parts: the gym's informational pages and the free fitness tools.

### üèãÔ∏è Gym Website Features
* **Pricing & Plans:** A clear, three-card layout displaying different membership tiers like "Gym Access," "Personal Training," and "Competition Training," with a checklist of included services.
* **Contact Form:** A "Get Connected with Gym" form that allows users to send their name, phone number, program interest, and a message.
* **Contact Information:** Displays the gym's physical address, phone number, email, and social media links (Facebook, Instagram).

### üßÆ Free Fitness Tools
* **Calorie Calculator:** A tool that calculates daily calorie needs based on:
    * Goal (e.g., Lean Gain, Weight Loss)
    * Weight (in kg)
    * Height (in cm)
    * Age (in years)
    * Gender
* **BMI Calculator:** A separate utility (`BMI.html`) to quickly calculate Body Mass Index.
* **Diet Plans:** A dedicated page (`dietPlan.html`) offering diet plans presumably based on different calorie ranges.
* **Workout Plans:** A section (`workoutPlan.html`) with various workout plans tailored to different fitness goals.

This website is built using fundamental web technologies, making it lightweight and easy to deploy.
